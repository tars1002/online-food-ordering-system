# Online Food Ordering System (DSA)

A command-line based Online Food Ordering System implemented in C to demonstrate core Data Structures and Algorithms through a real-world use case.

## Features
- Add and display restaurants
- Add and view menu items
- Place and process customer orders
- Persistent data storage using files

## Data Structures Used
- Hash Table (Separate Chaining)
- Linked Lists
- Queue (FIFO)

## Concepts Practiced
- Hashing
- Queues
- Linked Lists
- File Handling
- Problem-solving using DSA

## Future Enhancements
- GUI or web-based interface
- Database integration
- Multi-user support
